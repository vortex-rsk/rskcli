# rskcli
CLI for rsk

Available commands:

* clientVersion
* sha3 <arg>
* version
* listening
* peerCount
* peerList
* chainId
* ethProtocolVersion
* protocolVersion
* syncing
* coinbase
* mining
* hashrate
* gasPrice
* accounts
* blockNumber


## Install

1. Download the binary for your platform in the [releases page](https://github.com/rskcli/rskcli/releases)
put it on your path 
2. Put the file rskcli.conf from the root of the repo on your home folder
3. Be happy
